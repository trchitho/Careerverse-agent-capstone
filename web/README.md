# CareerVerse Agent Frontend

React + TypeScript + Vite web UI dashboard for visual career advisor demonstrations.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Dev server start:
   ```bash
   npm run dev
   ```

3. Build production output:
   ```bash
   npm run build
   ```

## Environment Variables

Configure `VITE_API_BASE_URL` if the FastAPI backend runs on a different port or host than `http://127.0.0.1:8000`.
