# Agent Skills

This directory contains reusable skill instructions for CareerVerse Agent.

Current skills:

- `career_advisor/SKILL.md`: career guidance workflow
- `code_quality/SKILL.md`: code review and quality checks
- `security_review/SKILL.md`: security and responsible AI review
- `kaggle_submission/SKILL.md`: Kaggle submission preparation

These files document how agents should perform project-specific tasks safely and consistently.
