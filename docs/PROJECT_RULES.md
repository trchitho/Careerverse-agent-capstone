# Project Rules — CareerVerse Agent

## Project Scope

CareerVerse Agent is a focused Kaggle Capstone MVP for educational career guidance.

It should demonstrate:

- Agent-based workflow
- Multi-agent orchestration
- MCP-style tool integration
- Agent Skills
- Security and responsible AI
- Local evaluation
- FastAPI deployability
- Clean documentation

## Non-Goals

This MVP must not become the full academic CareerVerse system.

Do not implement unless explicitly requested:

- Full RIASEC engine
- Full Big Five engine
- Real PhoBERT inference
- Real vi-SBERT embeddings
- Real NeuMF training
- Real Neo4j production graph
- Real pgvector database
- Payment
- Admin portal
- Full mentor matching
- Native mobile app
- Real-time video call
- Full cloud production deployment

## MVP User Journey

1. User submits education, interests, skills, and career goal.
2. Career Advisor Agent analyzes the profile.
3. Recommendation tool returns top career paths.
4. Skill Gap Agent compares user skills against required skills.
5. Roadmap Agent generates a learning roadmap.
6. API returns structured result with safety notice.

## Quality Target

The project should be:

- Easy to run
- Easy to demo
- Easy to understand
- Safe for public GitHub
- Suitable for Kaggle Capstone submission
- Honest about implemented vs future features
